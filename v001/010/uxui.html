DiaBeatIt App

An app to help prevent diabetes through health eating.

The app calculates and records daily calorie intake and has search tools for healthy eating.

DiaBeatIt is a MERN Stack app (MongoDB, Express, React, Node). The MongoDB database is hosted by mLab and the app is deployed on Heroku.

Technologies used: React, JavaScript, CSS, JSX, HTML, Bootstrap, Font Awesome, Datepicker, Moment, Node, Express, Mongoose, ORM (Object-Relational Mapping), MongoDB, NoSQL, RESTful API, Google Maps API, Yelp Fusion API, Edamam API, JSON, AXIOS, Passport, Google OAuth, mLab, Heroku.

App is responsive and designed for a mobile screen and larger.
